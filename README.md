# Generate-Binary-Numbers

Generate Binary Numbers
Given a number N, the task is to generate and print all binary numbers with decimal values from 1 to N.

Sample Input 1:
N = 2

Sample Output 1:
1 10

Sample Input 2:
N = 5

Sample Output 2:
1 10 11 100 101


Time Complexity  : o(n)
Space Complexity: o(n)
